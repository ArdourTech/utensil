# Tasks

Development tasks for Utensil

## deps
> Downloads and installs the necessary dependencies

~~~bash
lein deps
~~~

## test

> Runs the unit test suite

~~~bash
lein test
~~~

## install

> Builds and installs Utensil to the local repository

~~~bash
lein install
~~~

## deploy

> Builds and deploys Utensil to Clojars

~~~bash
lein deploy
~~~
